# algorithms-notes-byabdul-bari
# 🎓 Algorithms Study Tracker & Notes

This repository is dedicated to my journey through the **Abdul Bari Algorithms** course. It features an interactive **Python Dashboard** to manage my study progress and a collection of detailed notes for my **Master's in AI & Data Science**.

## 🚀 The Study Dashboard
I built a custom dashboard using **Streamlit** that allows me to:
* **Watch Lessons**: Integrated YouTube player for each video in the course.
* **Track Progress**: A dynamic progress bar that updates as I complete lessons.
* **Automated Notes**: A built-in editor that saves my insights directly into the `/notes` folder as Markdown files.



## 📂 Repository Structure
* `dashboard.py`: The main Python application for the study tracker.
* `Abdul Bari - Video List.json`: The source data containing video metadata.
* `requirements.txt`: List of dependencies needed to run the app.
* `/notes`: Automatically generated Markdown notes for each algorithm.
