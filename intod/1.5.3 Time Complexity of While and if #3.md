## 1.5.3 Time Complexity of While and if #3


### Core Concept

This video tackles more sophisticated logic, specifically how **conditional statements (if-else)** and **while loops** with varied jump conditions impact complexity. It also introduces the idea that not all loops are controlled by a simple $n$.

### Key Scenarios Analyzed

* **While Loops with Multiplication:** Similar to the previous video, a while loop where the iterator `i` is multiplied by a constant (e.g., `i = i * 3`) results in a logarithmic number of steps.
* **Complexity:** $O(\log_3 n)$.


* **The `if-else` Rule:** When analyzing an `if-else` block, you take the **maximum** complexity of the two branches.
* *Example:* If the `if` block is $O(1)$ and the `else` block is $O(n)$, the complexity of the entire structure is $O(n)$.


* **Nested Loops with Variable Limits:** Loops where the inner loop's execution depends on the outer loop's value in a non-linear way (e.g., `for(i=1; i<n; i=i*2)` containing an inner loop that runs $n$ times).
* **Complexity Calculation:** (Outer loop iterations) $\times$ (Inner loop iterations) = $(\log n) \times n$.
* **Result:** $O(n \log n)$.
